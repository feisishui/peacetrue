== 需求
导出后台列表数据。

== 设计
* 导出集合对象
* 每一列都有一个表头和转换器，转换器负责将对象转换成单元格内容

== 实现
* 导出``csv``格式
* 支持对象属性
* 支持对象嵌套属性
* 支持布尔型转换
* 支持日期型转换
* 支持单元格内容附加前缀防止格式错乱
* 支持自定义内容转换

== 使用方法
.引入类库
[source%nowrap,java]
----
<dependency>
    <groupId>com.github.peacetrue</groupId>
    <artifactId>peacetrue-export</artifactId>
    <version>1.0-SNAPSHOT</version>
